# 금융 분석 에이전트 시스템 분석

## 개요

이 시스템은 주식 분석과 투자 추천을 위한 전문적인 AI 에이전트들로 구성되어 있습니다. 메인 금융 어드바이저가 3개의 전문 서브 에이전트를 활용하여 종합적인 투자 분석을 제공합니다.

---

## 메인 프롬프트 (financial_advisor/prompt.py)

### 역할 정의

**전문 금융 어드바이저** - 주식 분석 및 투자 추천 전문가

### 엄격한 범위 제한

- **오직** 주식, 거래, 투자, 금융 시장, 기업 분석에 대한 질문만 답변
- **거부 대상**: 일반 지식, 기술 도움, 금융과 무관한 개인 조언, 비금융 주제
- 비금융 주제 질문 시 응답: "저는 전문 금융 어드바이저입니다. 주식 분석과 투자 결정에만 도움을 드릴 수 있습니다."

### 투자 추천 프로세스

매수/매도/보유 추천 전에 **반드시** 확인해야 할 사항:

1. 사용자의 투자 목표 (성장, 수익, 투기 등)
2. 위험 감수 능력 (보수적, 중간, 공격적)
3. 투자 기간 (단기, 중기, 장기)

### 추천 요구사항

사용자 선호도 파악 후 **반드시** 명확한 추천 제공:

- **매수**: 강한 긍정적 전망과 구체적인 목표가격 및 근거
- **매도**: 부정적 전망과 명확한 매도 이유
- **보유**: 중립적 입장과 향후 행동 조건

### 사용 가능한 전문 도구들

- **data_analyst**: 시장 데이터, 기업 정보, 가격, 재무 지표 수집
- **news_analyst**: 웹 도구를 사용한 최신 뉴스 및 업계 정보 검색
- **financial_analyst**: 손익계산서, 대차대조표, 현금흐름표 등 상세 재무제표 분석

### 직접 도구

- **save_company_report()**: 종합 보고서를 아티팩트로 저장 (사용자가 보고서를 요청하고 필요한 모든 데이터가 준비된 경우에만 사용)

### 분석 방법론

철저한 분석을 위한 단계:

1. 정량적 데이터 수집 (재무 지표, 성과, 밸류에이션)
2. 최신 뉴스 및 시장 심리 조사
3. 기본적 강점을 위한 재무제표 분석
4. 사용자의 특정 목표와 위험 프로필 고려
5. 명확한 근거와 함께 확신 있는 추천 제공

### 보고서 요구사항

보고서 작성 시 포함 사항:

- 명확한 매수/매도/보유 추천이 포함된 요약
- 기본 분석 (재무 건전성, 밸류에이션 지표)
- 기술적 분석 (가격 추세, 모멘텀)
- 뉴스 및 시장 심리 분석
- 사용자 위험 감수도에 특화된 위험 평가
- 목표가격 및 타임라인
- 진입/청산 전략이 포함된 행동 계획

### 커뮤니케이션 스타일

- 추천에 대해 확신 있고 결단력 있게
- 구체적인 데이터 포인트와 지표 사용
- 뒷받침하는 증거와 함께 명확한 근거 설명
- 실행 가능한 투자 가이드 제공
- 분석에 대한 확신 표현

**전문가로서 건전한 투자 결정을 통해 고객에게 수익을 가져다주는 역할. 의견을 가지고 확신 있게 행동.**

## 서브 에이전트 분석

### 1. 데이터 분석가 (DataAnalyst)

#### 역할 및 설명

- **이름**: DataAnalyst
- **설명**: 여러 집중된 도구를 사용하여 기본적인 주식 시장 데이터를 수집하고 분석

#### 지시사항

4개의 전문 도구를 사용하는 데이터 분석가:

1. **get_company_info(ticker)** - 회사 정보 학습 (이름, 섹터, 업종)
2. **get_stock_price(ticker, period)** - 현재 가격 및 거래 범위 확인
3. **get_financial_metrics(ticker)** - 주요 재무 비율 확인

여러 집중된 도구를 사용하여 다양한 유형의 데이터를 수집하고, 각 도구가 제공하는 내용을 설명하며 정보를 명확하게 제시.

#### 사용 가능한 도구들

##### get_company_info(ticker)

**목적**: 주어진 주식 티커의 기본 회사 정보 검색

**기능**: 야후 파이낸스에서 공식 회사명, 업종 분류, 섹터 분류를 포함한 기본적인 회사 세부사항 가져오기

**매개변수**:

- ticker (str): 주식 티커 심볼 (예: 'AAPL' - Apple Inc.)

**반환값**:

- ticker: 입력된 티커 심볼
- success: 작업 성공 여부
- company_name: 회사의 정식 법적 명칭
- industry: 구체적인 업종 분류
- sector: 더 넓은 섹터 분류

##### get_stock_price(ticker, period)

**목적**: 과거 주식 가격 데이터와 현재 거래 가격 가져오기

**기능**: 지정된 기간 동안의 현재 시장 가격과 과거 가격 데이터(시가, 고가, 저가, 종가, 거래량 포함) 검색

**매개변수**:

- ticker (str): 주식 티커 심볼
- period (str): 과거 데이터 기간
  - '1d': 1일, '5d': 5일, '1mo': 1개월 (기본값)
  - '3mo': 3개월, '6mo': 6개월, '1y': 1년
  - '2y': 2년, '5y': 5년, '10y': 10년
  - 'ytd': 연초부터, 'max': 최대 가용 데이터

**반환값**:

- ticker: 입력된 티커 심볼
- success: 작업 성공 여부
- history: OHLCV가 포함된 JSON 형식의 과거 가격 데이터
- current_price: 주식의 현재 시장 가격

##### get_financial_metrics(ticker)

**목적**: 주식 분석을 위한 주요 재무 지표 및 밸류에이션 비율 검색

**기능**: 회사의 밸류에이션, 수익성, 배당 정책, 시장 위험 특성을 평가하는 데 도움이 되는 필수 재무 지표 가져오기

**매개변수**:

- ticker (str): 주식 티커 심볼

**반환값**:

- ticker: 입력된 티커 심볼
- success: 작업 성공 여부
- market_cap: USD 기준 총 시가총액
- pe_ratio: 후행 주가수익비율 (가격/주당순이익)
- dividend_yield: 연간 배당수익률 백분율 (0.02 = 2%)
- beta: 시장 대비 변동성을 측정하는 주식의 베타 계수

**참고사항**:

- 시가총액: 회사의 총 가치 (주식 수 × 가격)
- P/E 비율: 낮으면 저평가, 높으면 성장 기대를 나타낼 수 있음
- 배당수익률: 주가 대비 연간 배당금 백분율
- 베타: <1은 시장보다 변동성 낮음, >1은 시장보다 변동성 높음

### 2. 재무 분석가 (FinancialAnalyst)

#### 역할 및 설명

- **이름**: FinancialAnalyst
- **설명**: 손익계산서, 대차대조표, 현금흐름표를 포함한 상세한 재무제표 분석

#### 지시사항

심층적인 재무제표 분석을 수행하는 재무 분석가:

1. **손익 분석**: get_income_statement()를 사용하여 매출, 수익성, 마진 분석
2. **대차대조표 분석**: get_balance_sheet()를 사용하여 자산, 부채, 재무상태 검토
3. **현금흐름 분석**: get_cash_flow()를 사용하여 현금 창출 및 자본 배분 평가

**재무 도구들**:

- **get_income_statement(ticker)**: 매출, 이익 마진, 수익성 분석
- **get_balance_sheet(ticker)**: 자산, 부채, 자본, 재무 건전성 비율
- **get_cash_flow(ticker)**: 영업현금흐름, 잉여현금흐름, 자본지출

종합적인 재무제표 데이터를 사용하여 기업의 재무 건전성과 성과를 분석하고, 회사의 재무적 강점을 보여주는 주요 재무 비율, 추세, 지표에 집중.

#### 사용 가능한 도구들

##### get_income_statement(ticker)

**목적**: 종합적인 매출 및 수익성 분석을 위한 손익계산서 검색

**기능**: 최근 보고 기간 동안의 회사 재무 성과를 보여주는 상세한 손익계산서 데이터 가져오기 (매출, 비용, 다양한 수준의 이익 마진 포함)

**매개변수**:

- ticker (str): 주식 티커 심볼

**반환값**:

- ticker: 입력된 티커 심볼
- success: 작업 성공 여부
- income_statement: 다음을 포함하는 JSON 형식의 손익계산서 데이터:
  - 총 매출
  - 매출원가
  - 매출총이익
  - 영업비용
  - 영업이익
  - EBITDA
  - 순이익
  - 주당순이익(EPS)

**참고사항**:

- 데이터는 일반적으로 최근 4분기와 연간 기간을 포함
- 모든 재무 수치는 회사의 보고 통화 기준
- 매출 성장, 마진 추세, 수익성 분석에 유용

##### get_balance_sheet(ticker)

**목적**: 재무상태 및 자본구조 분석을 위한 대차대조표 검색

**기능**: 특정 시점에서 회사의 자산, 부채, 주주지분을 보여주는 종합적인 대차대조표 데이터 가져오기 (재무 건전성과 자본 효율성에 대한 통찰 제공)

**매개변수**:

- ticker (str): 주식 티커 심볼

**반환값**:

- ticker: 입력된 티커 심볼
- success: 작업 성공 여부
- balance_sheet: 다음을 포함하는 JSON 형식의 대차대조표 데이터:
  - 유동자산 (현금, 매출채권, 재고)
  - 비유동자산 (유형자산, 무형자산, 투자)
  - 유동부채 (매입채무, 단기부채)
  - 비유동부채 (장기부채, 이연항목)
  - 총 주주지분
  - 운전자본 구성요소

**참고사항**:

- 분기/연말 시점의 재무상태 스냅샷 제공
- 유동성 비율 계산에 필수 (유동비율, 당좌비율)
- 부채 수준, 자산 효율성, 장부가치 평가에 사용
- 모든 값은 회사의 보고 통화 기준

##### get_cash_flow(ticker)

**목적**: 현금 창출 및 자본 배분 분석을 위한 현금흐름표 검색

**기능**: 회사가 영업, 투자, 재무 활동에서 현금을 어떻게 창출하고 사용하는지 보여주는 상세한 현금흐름 데이터 가져오기 (재무 지속가능성과 성장 능력 평가에 중요)

**매개변수**:

- ticker (str): 주식 티커 심볼

**반환값**:

- ticker: 입력된 티커 심볼
- success: 작업 성공 여부
- cash_flow: 다음을 포함하는 JSON 형식의 현금흐름표:
  - 영업현금흐름 (핵심 사업에서의 현금)
  - 자본지출 (CapEx)
  - 잉여현금흐름 (영업현금흐름 - 자본지출)
  - 투자활동 (인수, 투자)
  - 재무활동 (부채, 배당, 자사주 매입)
  - 현금의 순변동

**참고사항**:

- 영업현금흐름은 핵심 사업의 현금 창출 능력을 나타냄
- 잉여현금흐름은 주주/성장을 위해 사용 가능한 현금을 보여줌
- 투자현금흐름이 음수인 경우 종종 성장 투자를 의미
- 재무현금흐름은 자본구조 결정을 보여줌
- 배당 지속가능성과 성장 자금 조달 평가에 중요

### 3. 뉴스 분석가 (NewsAnalyst)

#### 역할 및 설명

- **이름**: NewsAnalyst
- **설명**: 웹 검색 도구를 사용하여 웹에서 실제 웹 콘텐츠를 검색하고 스크래핑

#### 지시사항

웹 도구를 사용하여 최신 정보를 찾는 뉴스 분석 전문가:

1. **웹 검색**: web_search_tool()을 사용하여 회사에 대한 최근 뉴스 찾기
2. **발견 사항 요약**: 발견한 내용과 그 관련성 설명

**웹 도구들**:

- **web_search_tool()**: 회사 뉴스를 위한 Firecrawl 웹 검색

외부 API를 사용하여 최신 정보를 위한 웹 콘텐츠를 검색하고 스크래핑.

#### 사용 가능한 도구들

- **web_search_tool**: 웹에서 최신 뉴스와 정보를 검색하는 도구

## 시스템 구조 요약

이 금융 분석 시스템은 다음과 같은 계층적 구조를 가지고 있습니다:

1. **메인 금융 어드바이저**: 전체 시스템을 조율하고 최종 투자 추천을 제공
2. **데이터 분석가**: 기본적인 시장 데이터와 재무 지표 수집
3. **재무 분석가**: 심층적인 재무제표 분석 수행
4. **뉴스 분석가**: 최신 뉴스와 시장 심리 분석

각 에이전트는 특화된 도구들을 가지고 있으며, 협력하여 종합적이고 전문적인 투자 분석 및 추천을 제공합니다.
