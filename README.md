# AI-agents

<br>

## Setup

Uses **`uv`** for Python environment & package management.

```
$ brew install uv
```

<br>

**1. Initialize a new agent**

```
$ uv init first-agent
```

<br>

**2. Sync dependencies**

check files: `.python-version`, `pyproject.toml`

```
$ cd first-agent
$ uv sync
```
